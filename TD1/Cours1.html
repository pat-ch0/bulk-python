Les chaines de caractères et print.

Pour construire une chaine de caractères, utilisez les délimiteurs/constructeurs : “” ou ‘’

“Ceci est une chaine de caractères.”
‘Ceci est aussi une chaine de caractères’

Pour utiliser le caractère ’ à l’intérieur d’une chaine : utiliser “”.
Pour utiliser le caractère " à l’intérieur d’une chaine : utiliser ‘’.
Voici quelques exemples.

print("Essai d'affichage d'une chaine")
print('Un autre "essai" pour voir la différence')

NB : print fait un passage à la ligne automatique à la fin d’exécution.

print("Pour forcer un passage à la ligne, utiliser '\\n'")
print("Pour contrôler vos passages à la ligne \n dans un print\n \n")

Pour afficher des valeurs variables dans une chaine : les f strings.

Attention : les f strings ne sont pas disponibles dans des versions un peu anciennes de Python.

Usage : faites précéder la chaine de la lettre f : f"…" et mettez entre {} les champs variables. Les {} ne feront pas partie de la chaine finales.



x = 4
print(f"La valeur de x est : {x} et son carré est {x**2}")

Ce print affiche à l’écran le message suivant :
La valeur de x est 4 et son carré est 16

Les strings sont des objets. Ils ont des méthodes associées. Essayez les lignes de code suivante pour voir ce que cela donne.
Création d’une chaine s :

s = "Bonjour M. Bond soyez le bienvenu chez nous"

Quelques exemples de méthodes utilisées sur la chaine s.

Attention : ces méthodes ne modifient pas la chaine s mais en construisent une nouvelle avec les bonnes caractéristiques.





print(f"s.upper() : {s.upper()}") # Pour tout mettre en majuscule
print(f"s.lower() : {s.lower()}") # Pour tout mettre en minuscule
print(f"s.title() : {s.title()}") # Chaque mot débute par une majuscule
print("Repetition 4 fois"*4)

split() : une méthode très intéressante pour obtenir la liste des mots.





l = s.split()
print(f'Avec s.split() la liste des mots de la chaine "{s}" est : \n {l}')
# Il est possible de "spliter" sur autre chose que les blancs
print(f'Spliter sur autre chose "a,b,,c,d".split(",") : {"a,b,,c,d".split(",")}')

La méthode join() est aussi très intéressante. Elle permet de créer une chaine de caractères à partir d’une liste de chaines de caractères (les listes sont vues dans une prochaine section).





l = ["Toto", "Tata", "Zoe"]
print(" Séparateur ".join(l))
print("".join(l))
print("*".join(l)) # Ici le résultat va être la chaine "Toto*Tata*Zoe"

Voici quelques exemples encore pour accéder aux éléments d’une chaine, de manière individuelle ou en “tranches” (slices).

Attention : le premier élément d’une chaine (ou d’une liste) est celui d’indice 0.

La fonction len(s) retourne la longueur de la chaine s (son nombre de caractères).







s = "0123456789"

# s[5]  :  caractère d'indice 5 de la chaine s
print(f'La chaine "{s}" est de longueur {len(s)}')
print(f'Le caractère d\'indice 0 de "{s}" est : {s[0]}')
print(f'Le caractère d\'indice 5 de "{s}" est : {s[5]}')

Syntaxe des slices [d:f:p] : d=début, f=fin, p=pas. Si le pas n’est pas spécifié, il est de 1 par défaut.

Attention : le caractère d’indice f n’est pas dans le slice.

Exemples d’utilisations de slices sur la même chaine s.







print(s[3:6]) # Par défaut le pas est de 1
print(s[6:]) # Ici c'est la sous-chaine depuis l'indice 6 jusqu'à la fin
print(s[:8]) # Depuis le début jusqu'à l'indice 7 (et pas 8)
print(s[::2]) # Ici le pas est 2
print(s[::-1]) # On peut afficher la chaine inversée comme ça.
print(s[-2:-5:-1]) # -2 = caractère d'indice 2 en partant de la droite de s.

ATTENTION ! une chaine est immuable (non modifiable). On ne peut pas faire s[6] = 9 par exemple.

Par contre une chaine est itérable. Dans le code suivant, la variable x reçoit succesivement les caractères de la chaine s (et les affiche grâce au print).



for x in s:
    print(x)

Cette forme est à préférer à :



for i in range(len(s)):
    print(s[i])

Il est possible de concaténer des chaines avec +

print(f'Concat de 2 chaines avec + : "Bonjour"+" Bye" = "{"Bonjour"+" Bye"}"')

Les listes

Les listes sont des objets modifiables et itérables.
Pour construire une chaine de caractères, utilisez les délimiteurs/constructeurs : [ et ]
Voici quelques lignes de code qui utilisent ces capacités.

l = [12, 55, 77]

Les listes peuvent contenir des éléments de “types” hétérogènes :






l = [-9, 7.77, "Bonjour vous", [1, 2]]
print(f"Voici le contenu de la liste l : {l}")

print(f"Son premier élément est à l'indice 0 : {l[0]}")
print(f"Sa longueur est : {len(l)}")

Une liste est un objet mutable (modifiable). Il est possible de modifier le contenu d’une case d’une liste



l[1] = -44
print(f"Après modification de l'élément d'indice 1, la liste est : {l}")

Pour ajouter un élément à la fin d’une liste, utilisez la méthode liste append() :





l = [1, 2, 3, 4, 5, 6]
print(f"Contenu de la liste l : {l}")
l.append(7)
print(f"La liste l après l.append(7) : {l} (insèré à la fin)")

Pour supprimer le dernier élément d’une liste, utilisez la méthode pop()







l = [1, 2, 3, 4, 5, 6]
l.pop()
print(f"La liste l après l.pop() : {l} (supprime le dernier)")
# Supprimer l'élément d'indice i avec l.pop(i) 
l.pop(3)
print(f"La liste l après l.pop(3) : {l} (supprime l'élément d'INDICE 3)")

Il est possible de concaténer deux listes avec +





l1 = [1, 2, 3, 4, 5, 6]
l2 = ["a", "b", "c"]
print(f"\nLe contenu des deux listes : l1={l1} et l2={l2}")
print(f"Résultat de l1 + L2 : {l1 + l2} (contact)")

Slices sur des listes : similaire à ce qui a été vu avec les strings

print(f"Slice sur la liste l1 : l1[2:5] = {l1[2:5]}")

Références de listes : attention !








print(f"\nRappel l1={l1}")
l2 = l1
print(f"l2 = l1. l2 référence la liste {l2}")
l1[1] = 999
print(f"Modification du contenu d'une case l1 : {l1}")
print(f"Impact sur l2 : {l2}")
print(f"l1 et l2 sont deux références sur le même objet")

Il est possible d’utiliser la méthode copy() pour faire une copie “physiquement” séparée.






l2 = l1.copy()
print(f"\nEn utilisant l2 = l1.copy(), l2 est : {l2}")
l1[1] = "Coucou c'est moi !"
print(f"Modifions l1 : {l1}")
print(f"Aucun impact sur l2 : {l2}")

Une liste est un objet itérable. Dans le code suivant, la variable x prend successivement les valeurs qui se trouvent dans la liste (et les affiche).




l = ["Bonjour", 17, 9.77, -42]
for x in l:
    print(x)

Les ranges

range(d,f) peut être assimilé à l’intervalle d’entiers [d,f[[d,f[

range(f) = range(0,f)

Un range est itérable. Dans le code suivant, la variable i prend les valeurs 0, 1, 2 (mais pas 3) et les affiche.



for i in range(3):
    print(i)

Attention : un range n’est PAS une liste. Un range est un… range.

Pour obtenir une liste à partir d’un range : list(…)



print("Transformer un range en liste : list(range(3, 10)) :")
print(list(range(3, 10)))

En pratique, les range sont souvent utilisés dans des boucles for.

Les sets (ensembles)

Pour créer un ensemble utiliser : { et } (comme en math). Un ensemble peut contenir des éléments de “types” hétérogènes.



s = {2, -9, 7.33, "Bond"}
print(f"Voici un ensemble référencé par s : {s}")

Un set est un objet itérable.




s = {2, -9, 7.33, "Bond"}
for x in s:
    print(x)

Dans un ensemble, un objet n’est présent qu’en un seul exemplaire.



s = {12+5, 'a', 17}
print("Voici l'ensemble : {12+5, 'a', 17} :" + f"{s}")

Ici s contient l’entier 17 et le caractère ‘a’.

Attention : un ensemble ne garde pas nécessairement l’ordre dans lequel les éléments ont été saisis.

Un set est mutable. Il est possible d’ajouter ou de supprimer un élément en utilisant des méthodes d’ensembles.







s = {1, 33, 8}
print(f"Le set s : {s}")
s.add(9) # Pour ajouter un élément dans l'ensemble s. 
print(f"Après ajout d'un élément avec la méthode s.add(...) : {s}")
s.discard(8) # Pour supprimer un élément dans l'ensemble s.
print(f"Après la suppression de l'élément 8 avec la méthode s.discard(8): {s}")

Je vous invite à essayer le code suivant.







l = list(range(5)) + [2, 1]
print(f"Voici une liste l : {l}")
print(f"La liste l transformée en set : set(l) : {set(l)}")
print("Dans le sens set -> liste avec list(...)")
s = {2, 5, 7, 8, 5}
print("list({2, 5, 7, 8, 5}) : " + f"{list(s)}")

Attention : pour créer un ensemble vide, utiliser set() et PAS {} (qui est un dictionnaire vide (voir plus loin)).

Les dictionnaires

Pour créer un dictionnaire, utilisez la syntaxe {… : …}




age = {"Albert" : 77, "Lola" : 15, "Kevin" : 20}
print(f"Le dictionnaire age est : {age}")
print(f"Le contenu de la case 'Lola' est age['Lola'] : {age['Lola']}")

Ici les indices de la structure ne sont pas nécessairement des entiers. C’est très pratique ! Le système détermine la case grâce à un procédé de hachage.

Attention : il faut que les indices soient immuables.

Un dictionnaire est mutable (modifiable).





age['Zoe'] = 33 # Pour ajouter un élément dans le dictionnaire
print(f"Le dictionnaire est maintenant : {age}")
del(age["Lola"]) # Pour supprimer un élément dans le dictionnaire
print(f"Le dictionnaire après suppression : {age}")

Un dictionnaire est itérable.







print("Itérons sur le dictionnaire")
for x in age:
    print(x)
# Pour obtenir le contenu des cases on peut faire :
for x in age:
    print(age[x])

On peut utiliser aussi les méthodes keys() (les indices) et values() (les valeurs).

Attention : ces méthodes ne donnent pas des listes mais des objets itérables.
Les deux boucles for précédentes peuvent aussi s’écrire :





for x in age.keys():
    print(x)
for x in age.values():
    print(x)

Les dictionnaires ont aussi une mthode items() qui donne un objet contenant les couples clés, valeurs sous forme de tuples (voir après).

Les tuples

Les tuples peuvent être vus, en première approximation, comme des listes non modifiables.
Syntaxe pour construire un tuple : ( et )




t = (2, -77.8, "Clermont")
print(f"Voici un tuple t : {t}")
print(f"t[1] : {t[1]}")

Un tuple à un seul élément 14 par exemple n’est PAS (14) mais (14,). Attention, (14) vaut 14 et PAS le tuple (14)

Un tuple est immuable ; On ne peut pas faire t[1] = 22 par exemple.
Oui MAIS, je vous invite à taper le code suivant et à essayer de comprendre ce qui se passe :







l = [1, 2]
t = ("a", l, 8) # La liste l est un élément du tuple
print(f"l : {l} t = ('a', l, 8) : {t}")
l.append(3)
print(f"Modifions l avec l.append(3) : {l}")
print(f"Impact sur le tuple t : {t}") # Le contenu du tuple est modifié

Un tuple est itérable




t = (1, 2, 3)
for x in t:
    print(x)

Il est possible, connaissant la taille d’un tuple, de récupérer ses composantes en une seule affectation




t = (1, 2, 3)
a, b, c = t
print(f"L'affecation a, b, c = t donne a={a}, b={b}, c={c}")

Attention : un tuple est défini même sans les paranthèses : 1, 2, 3 est le tuple (1, 2, 3).
Conseil : mettez les ( et ) pour clarifier.

La taille du tuple t est son nombre d’éléments et peut être obtenu par len(t)
Slice sur les tuples : t[0:2].
Transformer un tuple en liste list(t).
Transformer une liste en tuple avec tuple(l).

Pour accéder directement aux composantes de tuples stockés dans une liste l. Je vous invite à tester le code suivant et essayer de comprendre le mécanismes (très pratique).
N.B. : str(x) est une fonction qui retourne la chaine composé des caractères de l’entier x. Par exemple str(42) construit la chaine “42”.





l = [("x"+str(i), "y"+str(i), "z"+str(i)) for i in range(3)]
print(f"Accéder aux composantes de chaque tuple de la liste l {l}") 
for a, b, c in l:
    print(a + b + c)

Les compréhensions

Façon de construire une structure (une liste ou un set par exemple) de manière syntaxiquement “compacte”.

Construire une liste avec une classique boucle for :




l = []
for x in range(99,106):
    l.append(chr(x))  # chr(x) = caractère numéro x dans la table

Ici LA manière pythonique (“à la python”) de faire est la suivante :

l = [chr(x) for x in range(99,106)]

Il est possible d’imbriquer (attention à la lisibilité et à l’ordre des boucles !)

l = [chr(x) for y in range(1,4) for x in range(97,97 + y)]

La même chose mais pour construire un ensemble (set).

s = {chr(x) for y in range(1,4) for x in range(97,97 + y)}

Application : construire la liste des caractères d’une chaine avec une compréhension.



s = "abcd"
l = [x for x in s]

Les conditionnelles

Les entiers pairs entre 0 et 8, calculés et affichés avec une boucle for et if :




for i in range(9):
    if i % 2 == 0: 
        print(i)

Indication : a % b représente a modulo b, c’est-à-dire le reste de la division.
Le quotient (entier) de la division de a par b est a//b

Illustration de la syntaxe avec un exemple. Notez qu’il n’y a pas d’indentation cumulée (elif au même niveau que le if)




















print("if: ... elif ... elif ... else")
note = 13
print(f"Votre note est {note}. Voici votre mention :")
if note < 0:
    print(f"Erreur : note {note} négative !")
elif note < 10:
    print("Echec")
elif note < 12:
    print("Passable")
elif note < 14:
    print("AB")
elif note < 16:
    print("B")
elif note < 18:
    print("TB")
elif note <= 20:
    print("Félicitations !")
else:
    print(f"Erreur, note {note} > 20 impossible")

Le if… else sous forme d’expression (le else est obligatoire).
Exemple : élever x au carré s’il est pair et au cube sinon.





x = 9

print(f"x**2 if x % 2 == 0 else x**3 : {x**2 if x % 2 == 0 else x**3}")

Voila le bon endroit pour introduire les Booléens : True et False (notez les majuscules). Python a des opérateurs logiques : or, and, not…

Les fonctions

Le mot clé pour définir une fonction : def




def f(x, y, z):
    result = x + y + z
    print(f"La somme des paramètres d'entrée {x, y, z} est : {result}")
  • f est le nom de la fonction. ATTENTION : donnez des noms qui ont un sens.
  • x, y, z sont les 3 paramètres formels de cette fonction.
  • result est une variable locale (non connue à l’extérieur de f).
  • Le reste est le corps de la fonction, un bloc d’instructions.

Notez au passage : pass = instruction qui ne fait rien.

Utilisation de la fonction f :



f(1, 2, 3)
f('Toto', ' a ', 'perdu')

Ici les paramètres passés sont souvent appelés les paramètres effectifs.
La valeur de l’appel d’une fonction est… ça dépend :

  • S’il n’y a pas de return, c’est la valeur None.
  • Sinon c’est la valeur du premier return rencontré lors de l’exécution.

Le premier return rencontré retourne la valeur associée et stoppe l’exécution de la fonction.









def g(x):
    if x%2 == 0:
        return "Pair"
    else:
        return "Impair"
    
print(f"Valeur de g(4) : {g(4)}")
print(f"Valeur de g(7) : {g(7)}")

Une fonction peut définir et utiliser une (ou plusieurs) fonction locale









def message_parite(x):
    """ On peut placer une documentation de la fonction (docstring).
        Celle-ci peut tenir sur plusieurs lignes.
        help(nom_de_la_fonction) pour l'afficher.
        Ceci est une pratique recommandée pour documenter son code."""
    def parite(x):
        return "Pair" if x%2 == 0 else "Impair"
    print(f"Le paramètre {x} est {parite(x)}")

Notez : la fonction locale parite() retourne une valeur mais PAS la fonction message_parite qui ne fait qu’afficher un message.

L’indentation du code, règles d’écriture du code

En python, un bloc d’instructions est délimité par l’indentation des instructions qu’il contient. Ne changez pas l’indentation par défaut de Idle, sinon votre code ne sera plus valide pour d’autres utilisateurs. Voici un exemple.






print("Bonjour")
for i in range(3):
    print(i)
    print("Dans la boucle")
print("FIN")

Ce code va afficher :

Bonjour
0
Dans la boucle
1
Dans la boucle
2
Dans la boucle
FIN

Alors que le code suivant :






print("Bonjour")
for i in range(3):
    print(i)
print("Dans la boucle")
print("FIN")

va afficher :

Bonjour
0
1
2
Dans la boucle
FIN

En Python, le nom des fonctions et des variables doit être explicite, cela participe à la lisibilité du code. Ces identifiants doivent être écrits en minuscule, les mots étant séparés par _
Par exemple : liste_des_numéros (notez l’accent ici), ensemble_des_noms, affiche_les_prix.
Après une virgule, il est d’usage de laisser un espace. Par exemple, pour former un tuple, écrire :

(5, 6, 7)

plutôt que :

(5,6,7)

En Python, chaque ligne ne doit pas depasser 79 caractères.

De nombreuses autres conventions existent et font partie du PEP 8 de Python mais je vous demande explicitement de respecter ces quelques règles qui améliorent considérablement la lisibilité (et du coup la maintenabilité) du code.

Voici un exemple de mise en oeuvre de ces bonnes pratiques.
Voici ce qu’il ne faut pas écrire :






def f(a,b,c):
    print(f"Voici l'identité de la personne : \nNom : {a[0]}, prénom : {a[1]}, Marié ? {'Oui' if b else 'Non'}, age : {c}")


f(("James","Bond"),False,55)

Ici les lignes sont trop longues, les noms ne sont pas explicites. Code difficile à lire.
On préfèrera :









def affiche_identité(nom_prénom, est_mariée, age):
    nom, prénom = nom_prénom
    print(f"Voici l'identité de la personne :")
    print(f"Nom : {nom}, prénom : {prénom}, " \
          f"Marié ? {'Oui' if est_mariée else 'Non'},"\
          f"age : {age}")

affiche_identité(("James", "Bond"), False, 55)
